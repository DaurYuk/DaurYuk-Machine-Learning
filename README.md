## About the Model

The model is made to classify waste object into nine different classes:

1. Cardboard
2. Food organics
3. Glass
4. Metal
5. Miscellaneous trash
6. Paper
7. Plastic
8. Textile trash
9. Vegetation

**Link to dataset:** https://www.kaggle.com/datasets/joebeachcapital/realwaste/data
